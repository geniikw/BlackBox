이번달 대한민국게임잼 2016에서 만든 프로젝트입니다. 
3일간 게임잼버전을 일주일간 혼자서 리펙토링한 버전입니다.
내부에 포함된 모든 모듈과 코드는 제가 짠 코드입니다.
기획,코딩을 제가 했고 UI나 아이콘등의 디자인은 게임잼 팀원이 했습니다.
